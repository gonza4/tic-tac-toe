Quick start
-----------

- Clone the repo: `git clone https://github.com/gonza4/tic-tac-toe.git`

- Install dependencies: `npm install`
- Run develop server: `npm start`

- Move to ttt-front: `cd ttt-front`
- Start server: `ng serve --o`
